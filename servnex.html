Technical Architecture

ServNex utilizes a modern decoupled architecture. The Django REST Framework backend serves as a robust engine handling complex business logic, while the React frontend provides a seamless, app-like experience for users.

  • 01.
    Google OAuth 2.0

    Secure and frictionless social authentication for improved user retention.

  • 02.
    Razorpay API Integration

    End-to-end payment lifecycle management including signature verification and failure reporting.

  • 03.
    Signal-based Notifications

    Automated triggers for booking confirmations, cancellations, and payment reminders.

Overcoming the "Payment Mismatch" Challenge

One of the biggest hurdles was ensuring data consistency when users cancelled payments mid-process. I implemented a robust "Post-Payment Verification" system and centralized failure logging to ensure that no booking remained in a "Ghost" state, providing clear feedback to users even when transactions failed.
